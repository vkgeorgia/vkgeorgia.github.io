# План разработки цифрового аватара (Текстовый AI-консультант)

## Цель проекта
Создание умного текстового чат-бота ("цифрового двойника"), который:
1.  Отвечает на вопросы рекрутеров и клиентов, используя персональную базу знаний (резюме, кейсы, услуги).
2.  Работает в реальном времени через веб-интерфейс.
3.  Интегрируется в личный веб-сайт (`vkgeorgia.github.io`).
4.  Автоматизирует запись на встречи через Google Calendar Appointment Schedules.
